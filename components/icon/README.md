# Icon Component
A simple component that wraps <FontAwesomeIcon> component.

### props
- **_`icon`_**: string
  - Specifies the icon
- **_`size`_**: string
  - Specifies the fontsize of the icon

 
### Usage

```javascript
import Icon from "@comp/icon/"
import { faCoffee } from '@fortawesome/free-solid-svg-icons'

<Icon
  icon={faCoffee}
  size="24"
/>
```
