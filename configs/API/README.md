# APIS

There are three different kinds of enpoints

- **_`delivery`_**
  - Specifies the published contents : _delivery.blupoint.io_
- **_`preview`_**
  - Specifies the unpublished contents : _preview.blupoint.io_
- **_`management`_**
  - Only for updating, deleting and inserting something... : _management.blupoint.io_
